Technical Approach

We decided to use the Navier-Stokes equation as a basis for our fluid simulation.


The Navier-Stokes equation can be broken up into four main parts, advection, pressure, viscosity and external force.

Problems

Once our prototype was prepared, we ran into an issue with p5.js in which we were unable to pass negative numbers between fragments. We attempted to use a sigmoid function to remedy this, but this led to further issues. We ultimately decided to swap from using p5.js to regular JavaScript in order to get around this issue.

Lessons Learned

Debugging in WebGL and p5.js is quite difficult when attempting to read in values of certain variables when working with shaders. Writing a rough prototype in python where it is much easier to debug proved very useful for our final implementation.
